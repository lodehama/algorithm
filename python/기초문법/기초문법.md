#### **자료형 (data-types)**

숫자형

\- int : 정수

\- float : 실수

\- complex : 복소수



문자열

\- str



불린형

\- bool (True, False)

#### 

#### **입출력 (input-output)**

입력

\- input() : 사용자로부터 값을 입력받음



출력

\- print() : 값을 화면에 출력



자료형 확인

\- type() : 변수의 자료형을 확인



### **반복문**

\- for : 정해진 횟수만큼 반복

\- while : 조건이 참인 동안 반복



반복 제어

\- break : 반복문 종료

\- continue : 다음 반복으로 이동



반복 범위

\- range() : 연속된 숫자 생성



#### **함수 (functions)**

\- def : 함수 정의



매개변수

\- parameter : 함수 정의 시 사용하는 변수



인자

\- argument : 함수 호출 시 전달하는 값



반환값

\- return : 함수 실행 결과 반환



내장 함수

\- print()

\- input()

\- len()

\- type()
